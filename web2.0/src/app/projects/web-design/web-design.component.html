EAST Initiative Website Redesign

I created this wireframe for Kevin Cate's "Web Design" class. This course was taught with a focus on the Graphic Designing element of building websites. We explored how to place content in a way that was eye catching and simultaneously easily usable.

For this specific project, we were tasked with redesigning the entire website of an existing non-profit business. I chose to redesign eastinitiative.org. I tried to keep the original intention of the design while modifying elements such as the color palette and the layout for a fresh look.
